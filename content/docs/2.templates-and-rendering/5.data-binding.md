# Data Binding

`v-bind` (shorthand `:`) dynamically binds attributes, classes, styles, and props to expressions.

## Basic Attribute Binding

```vue
<script setup>
const imageUrl = ref('/photo.jpg')
const linkUrl = ref('https://vuejs.org')
const inputId = ref('email-input')
</script>

<template>
  <img :src="imageUrl" :alt="'User photo'">
  <a :href="linkUrl">Vue.js</a>
  <input :id="inputId" type="email">
</template>
```

### Boolean Attributes

Boolean attributes are included/excluded based on truthiness:

```vue
<template>
  <button :disabled="isSubmitting">Submit</button>
  <input :readonly="!isEditable">
  <option :selected="isDefault">Default</option>
</template>
```

## Class Binding

### Object Syntax

```vue
<script setup>
const isActive = ref(true)
const hasError = ref(false)
</script>

<template>
  <div :class="{ active: isActive, 'text-error': hasError }">
    Styled div
  </div>
  <!-- Renders: <div class="active"> -->
</template>
```

### Array Syntax

```vue
<script setup>
const baseClass = ref('card')
const sizeClass = ref('card-lg')
</script>

<template>
  <div :class="[baseClass, sizeClass]">Card</div>
  <!-- Renders: <div class="card card-lg"> -->

  <!-- Mix with conditional -->
  <div :class="[baseClass, { active: isActive }]">Card</div>
</template>
```

### With Computed

```vue
<script setup>
const type = ref('success')

const alertClass = computed(() => ({
  'alert': true,
  'alert-success': type.value === 'success',
  'alert-error': type.value === 'error',
  'alert-warning': type.value === 'warning',
}))
</script>

<template>
  <div :class="alertClass">{{ message }}</div>
</template>
```

## Style Binding

### Object Syntax

```vue
<script setup>
const color = ref('red')
const fontSize = ref(16)
</script>

<template>
  <p :style="{ color: color, fontSize: fontSize + 'px' }">
    Styled text
  </p>

  <!-- camelCase or kebab-case both work -->
  <p :style="{ 'font-size': fontSize + 'px' }">Text</p>
</template>
```

### Array Syntax

Merge multiple style objects:

```vue
<script setup>
const baseStyles = { color: 'blue', fontWeight: 'bold' }
const layoutStyles = { padding: '10px', margin: '5px' }
</script>

<template>
  <div :style="[baseStyles, layoutStyles]">Merged styles</div>
</template>
```

### Auto-prefixing

Vue automatically adds vendor prefixes when needed:

```vue
<template>
  <div :style="{ display: 'flex' }">
    <!-- Vue adds -webkit-flex etc. if needed -->
  </div>
</template>
```

## Binding Multiple Attributes

Bind an object of attributes at once (no argument needed):

```vue
<script setup>
const inputAttrs = {
  id: 'username',
  type: 'text',
  placeholder: 'Enter username',
  required: true,
}
</script>

<template>
  <input v-bind="inputAttrs">
  <!-- Equivalent to: <input id="username" type="text" placeholder="Enter username" required> -->
</template>
```

## Dynamic Attribute Names

Use bracket syntax for dynamic attribute names:

```vue
<script setup>
const attrName = ref('href')
const attrValue = ref('https://vuejs.org')
</script>

<template>
  <a :[attrName]="attrValue">Link</a>
</template>
```

## Same-name Shorthand (Vue 3.4+)

When the attribute name matches the variable name:

```vue
<script setup>
const id = ref('my-input')
const type = ref('text')
</script>

<template>
  <!-- Shorthand — new in Vue 3.4 -->
  <input :id :type>

  <!-- Equivalent to -->
  <input :id="id" :type="type">
</template>
```

## Common Pitfalls

| Pitfall | Solution |
|---|---|
| Forgetting `:` prefix | `href="url"` is a static string, `:href="url"` is dynamic |
| Style values without units | Always include units: `fontSize + 'px'` |
| Mutating class objects in-place | Use computed or return new objects |
| kebab-case in style objects | Use camelCase (`fontSize`) or quote it (`'font-size'`) |

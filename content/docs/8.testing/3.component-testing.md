# Component Testing

`@vue/test-utils` provides utilities to mount Vue components in isolation and test their behavior.

## Setup

```bash
npm install -D @vue/test-utils happy-dom
```

## Mount vs ShallowMount

```ts
import { mount, shallowMount } from '@vue/test-utils'
import MyComponent from './MyComponent.vue'

// mount — renders child components too
const wrapper = mount(MyComponent)

// shallowMount — stubs all child components
const shallow = shallowMount(MyComponent)
```

**Use `mount`** for integration-style tests. **Use `shallowMount`** to isolate a component from its children.

## Testing Props

```vue
<!-- Greeting.vue -->
<script setup>
defineProps<{ name: string; greeting?: string }>()
</script>
<template>
  <p>{{ greeting ?? 'Hello' }}, {{ name }}!</p>
</template>
```

```ts
describe('Greeting', () => {
  it('renders with name', () => {
    const wrapper = mount(Greeting, { props: { name: 'Alice' } })
    expect(wrapper.text()).toContain('Hello, Alice!')
  })

  it('uses custom greeting', () => {
    const wrapper = mount(Greeting, { props: { name: 'Bob', greeting: 'Hi' } })
    expect(wrapper.text()).toContain('Hi, Bob!')
  })
})
```

## Testing Events

```vue
<!-- Counter.vue -->
<script setup>
const emit = defineEmits<{ increment: [count: number] }>()
const count = ref(0)

function handleClick() {
  count.value++
  emit('increment', count.value)
}
</script>
<template>
  <button @click="handleClick">Count: {{ count }}</button>
</template>
```

```ts
describe('Counter', () => {
  it('emits increment event', async () => {
    const wrapper = mount(Counter)
    await wrapper.find('button').trigger('click')

    expect(wrapper.emitted('increment')).toBeTruthy()
    expect(wrapper.emitted('increment')![0]).toEqual([1])
  })

  it('increments on multiple clicks', async () => {
    const wrapper = mount(Counter)
    await wrapper.find('button').trigger('click')
    await wrapper.find('button').trigger('click')

    const events = wrapper.emitted('increment')!
    expect(events).toHaveLength(2)
    expect(events[1]).toEqual([2])
  })
})
```

## Finding Elements

```ts
const wrapper = mount(MyComponent)

// By CSS selector
wrapper.find('button')
wrapper.find('.my-class')
wrapper.find('#my-id')
wrapper.find('[data-testid="submit"]')

// Find all
wrapper.findAll('li')

// Find component
wrapper.findComponent(ChildComponent)

// Check existence
expect(wrapper.find('.error').exists()).toBe(false)
```

## Testing v-model / Input

```ts
it('updates on input', async () => {
  const wrapper = mount(SearchInput)
  const input = wrapper.find('input')

  await input.setValue('hello')
  expect(wrapper.vm.query).toBe('hello')
})
```

## Testing Slots

```vue
<!-- Card.vue -->
<template>
  <div class="card">
    <header><slot name="header" /></header>
    <main><slot /></main>
    <footer><slot name="footer" :count="items.length" /></footer>
  </div>
</template>
```

```ts
describe('Card', () => {
  it('renders default slot', () => {
    const wrapper = mount(Card, {
      slots: { default: 'Main content' }
    })
    expect(wrapper.find('main').text()).toBe('Main content')
  })

  it('renders named slots', () => {
    const wrapper = mount(Card, {
      slots: {
        header: '<h1>Title</h1>',
        default: 'Body',
      }
    })
    expect(wrapper.find('header').html()).toContain('Title')
  })

  it('renders scoped slots', () => {
    const wrapper = mount(Card, {
      slots: {
        footer: ({ count }) => `Total: ${count}`,
      }
    })
    expect(wrapper.find('footer').text()).toContain('Total:')
  })
})
```

## Testing Async Behavior

```ts
it('shows data after fetch', async () => {
  const wrapper = mount(UserList)

  // Wait for async operations
  await flushPromises()

  expect(wrapper.findAll('li')).toHaveLength(3)
})

it('shows loading state', () => {
  const wrapper = mount(UserList)
  expect(wrapper.find('.loading').exists()).toBe(true)
})
```

```ts
  }

```

## Testing with nextTick

After reactive state changes, the DOM updates asynchronously:

```ts
it('updates DOM after state change', async () => {
  const wrapper = mount(Counter)

  wrapper.vm.count = 5
  await wrapper.vm.$nextTick()  // or: await nextTick()

  expect(wrapper.text()).toContain('5')
})
```

`trigger()` and `setValue()` already include `await nextTick()`.

## Best Practices

1. **Test behavior, not implementation** — test what the user sees and does
2. **Use `data-testid`** for stable selectors
3. **Avoid testing internal state** — test through the rendered output
4. **One assertion per concept** — not per test, but keep focus clear
5. **Mount only what you need** — use stubs for unrelated children

## Common Pitfalls

| Pitfall | Solution |
|---|---|
| DOM not updated after state change | `await nextTick()` or use `trigger()` which auto-awaits |
| Child component not rendering | Check if using `shallowMount` (stubs children) |
| `find()` returns empty | Check selector, ensure element is rendered (not `v-if`'d out) |
| Global plugins not available | Pass in `global.plugins` option |
